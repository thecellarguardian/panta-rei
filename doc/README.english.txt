=========
Panta Rei
=========
Panta Rei is a real-time scheduling simulator.
The current version provides both an interactive CLI 
interface and a batch script based one.
Panta Rei allows to simulate the scheduling of periodic
task sets through the Rate Monotonic, Deadline Monotonic
and Earliest Deadline First algorithms, with or without
preemption. 
With Panta Rei you can view the simulation through graphs
and set simulation and error logs.

============
Requirements
============
System: built and tested under GNU/Linux
Dependencies: gnuplot

=====
Usage
=====
You can display a list of program options launching
the application with the "-h" option.
For a complete script language description, use the
interactive command "syntax".
